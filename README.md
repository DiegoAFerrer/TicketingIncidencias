# TicketingIncidencias
Sistema de Ticketing de Incidencias en el Laboratorio de Computación

Este sistema se ejecuta desde la terminal.
Contiene una serie de menús que permiten reportar incidencias en el laboratorio de cómputo, así como ver por tipo y estado, asignar tecnicos a reparar y demás funciones. 
Se organiza en algunos menús y submenús para utilizar todas estas herramientas.
